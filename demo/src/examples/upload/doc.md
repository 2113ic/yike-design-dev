## YkUpload 文件上传

:::snippet
基本用法
文件列表上传
<UploadPrimary/>
:::

:::snippet
单文件上传
通过 `multiple` 控制是否能够上传多文件，默认为 true
<UploadSingleFile/>
:::

:::snippet
文件数量限制
通过 `limit` 控制文件上传数量限制
<UploadLimit/>
:::

:::snippet
图片上传
通过 `accept` 判断是否仅上传图片，若接受对象包含 `image` 、 `png` 、 `jpg/jpeg` ，则采用图片列表上传 UI
<UploadPicture/>
:::
