### API

#### Scrollbar Attributes

| 参数   | 描述           | 类型             | 默认值 |
| ------ | -------------- | ---------------- | ------ |
| height | 容器高度       | number ｜ string | --     |
| always | 滚动条总是显示 | boolean          | false  |

#### Scrollbar Slots

| 参数    | 描述       |
| ------- | ---------- |
| default | 自定义内容 |

#### Scrollbar Events

| 事件名 | 说明     | 类型     |
| ------ | -------- | -------- |
| scroll | 触发滚动 | Function |
